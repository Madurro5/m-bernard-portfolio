📡 Système de Test Vibratoire

Modernisation d'un Pot Vibrant LDS V726

STM32 Node.js SWEPT SINE Temps réel

Remise en service et modernisation complète d'un pot vibrant haute précision (valeur >10k€) en remplaçant un système de contrôle des années 1980 (SI1215) par une solution moderne basée sur STM32H755ZI et une interface web Node.js.

  • Architecture dual-core : Gestion Coeur M4 et M7 sur STM32
  • NCO avec accumulateur de phase : Génération du sinus avec un contrôle précis de la fréquence
  • Gestion synchrone DAC/ADC : Pour un contrôle d'échantillonnage
  • Communication USART : entre la carte STM32 et l'interface PC
  • Mode SWEPT SINE : Sinus à fréquence balayée avec amplitude contrôlée

Résultats obtenus :

✓ Stimulation SWEPT SINE ✓ Suivi fréquence précis 🔄 Suivi accélération 🔄 Réduction bruit

🖨️ Fraiseuse CNC PCB

Automatisation d'une CNC pour Fabrication PCB

Python FlatCAM GRBL MkDocs Electronique Automatisation

Automatisation complète du workflow de fabrication de circuits imprimés par création d'un plugin Python au sein du logiciel Open Source FlatCAM, réduisant le temps de préparation de plusieurs dizaines de minutes à quelques clics, avec documentation complète pour la pérennité du système.

  • Plugin Python natif intégré à FlatCAM pour automatisation complète
  • Conversion Gerber → G-Code sans paramétrage manuel
  • Gestion automatique des symétries, changements d'outils et perçages
  • Détection de surface pour mise à zéro Z précise (modification circuit électronique et paramétrage GRBL)
  • Documentation complète avec MkDocs (procédures, utilisation, reprise du développement)

Résultats obtenus :

✓ Temps réduit de 90% ✓ Élimination des erreurs ✓ Mise à zéro Z fiable ✓ Documentation maintenable ✓ Possibilité d'adapter les usinages
